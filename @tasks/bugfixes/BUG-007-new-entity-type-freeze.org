** DONE Use $state.snapshot() for yaml.stringify in SchemaEditor
:PROPERTIES:
:CUSTOM_ID: ITEM-907
:EFFORT: 15m
:PRIORITY: #A
:FILES: src/lib/components/entities/SchemaEditor.svelte
:END:

*** Description
~$derived(yaml.stringify(localSchema))~ traverses the full deep proxy, creating
broad reactive subscriptions. Use ~$state.snapshot()~ to convert to plain object
before passing to yaml. Also remove the ~state_referenced_locally~ suppression comment.
Additionally, ~structuredClone(schema)~ fails on $state proxy props — use
~$state.snapshot()~ before cloning. And ~handleSave~ was passing a proxy to
~onSave~ callback — use ~$state.snapshot()~ instead of ~structuredClone~.

*** Acceptance Criteria
- [X] yamlPreview uses ~$state.snapshot(localSchema)~ for yaml.stringify
- [X] ~state_referenced_locally~ suppression comment removed
- [X] ~structuredClone~ uses ~$state.snapshot(schema)~ for initial clone
- [X] ~handleSave~ uses ~$state.snapshot(localSchema)~ instead of ~structuredClone~
