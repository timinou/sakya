** Dashboard Commands

| Command | Keybinding | Description |
|---------+------------+-------------|
| =prd-dashboard-refresh= | =C-c p r= | Refresh all metrics |
| =prd-list-blocked= | =C-c p b= | Show blocked items |
| =prd-next-item= | =C-c p n= | Jump to next available task |
| =prd-velocity-report= | =C-c p v= | Generate velocity report |
| =prd-burndown-chart= | =C-c p B= | Generate burndown chart |

** CLI Commands (for Claude Code)

#+BEGIN_SRC bash
# Get dashboard as JSON
emacsclient -s sakya -e '(prd-dashboard-cli :format json)'

# Get blocked items
emacsclient -s sakya -e '(prd-list-blocked :format json)'

# Get velocity metrics
emacsclient -s sakya -e '(prd-velocity-report :days 14 :format json)'
